←EZPath→

EZPath helps you find the shortest way to travel between a set of destinations.

All you have to do is

  1. Enter where you would like to start and return to
  2. Enter each address you would like to travel to
  3. Click "Look Up Addresses" before solving to verify the addresses
  4. Click "Solve It!"
  5. Click the link to your map

Mode:
  1. Home Address (start and return here)
  2. Destinations
  5. No map link generated yet
